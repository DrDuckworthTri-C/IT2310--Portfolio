Welcome to Lab 2

In this lab, I will learn how to build a multi-page website with working navigation between pages. I'll practice creating hyperlinks using relative paths to connect my own pages together and absolute paths to link to external websites. I'll also build HTML tables with proper accessibility features like captions, header scopes, and semantic groupings that help screen readers understand my content. These navigation and structure skills will directly prepare me for the Campus Resource Roulette capstone project, where I'll need to connect multiple pages including a main wheel page, individual agent pages, and an about page.
